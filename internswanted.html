Using Geolocation as a Method of Fraud Detection

 

Recently, I tweeted that an idea for detecting fraud would be to use the geolocation technology in cell phones to check for where the person who owns the credit card is and compare it against where the transaction happened. This seems like a good idea, but also could have some issues. I'll talk about both.

First off, the benefits and the parts that work well. This ideas is based upon the idea that we all carry our mobile phones with us just about everywhere we go. We have it with us at all times of the day. So we can use the cell phone to locate the person. Once we have the person's location via the phone, we can check the location with the location of the transaction. This would work well with stores and shops, where the person is physically present in the store making a purchase. This is a convenient way to check that the person is actually in the store buying the goods. This can be done on every purchase, so that no matter how small or large the purchase is, it can be verified geographically. This could provide a faster recognition of fraud.

There are some issues with just checking the geographic location. A lot of times today we shop online, which makes verifying a geographic location more difficult. We aren't sure where the user is when they are purchasing things through the sites. This prevents a problem, because although we could check the person's cell phone location, we don't know where the transaction occurred to begin with! A solution to this is to have the online site to have the online site send the IP address of the computer on which the transaction took place along to the credit card company, who can then look up where the IP is located. This can used as a crude method of locating the transaction. However, IP lookups aren't always accurate, and could cause more confusion than necessary.

Another issue with online shopping is many people can use that same credit card for legitimate means. This comes up a lot in families, where the children use their parents' credit cards for online shopping. This can cause issues when the children and parents aren't in the same geographic location. For example, the child could be at college and make a purchase on the parents card. This can cause issues with using geographic location as a means for fraud detection because this scenario might raise a red flag, but in reality it's fine.

This issues can be resolved using more traditional means, such as a phone call to double check the card holder is aware of the purchase, or using other methods of fraud detection already in place.

Using geolocation of cell phones isn't going to replace all other methods of fraud detection, but would be there simply to complement the existing measures already established.
